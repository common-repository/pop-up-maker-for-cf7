This plugin expands the capabilities of contact-forms-7

== Description ==
Provides the ability to display a modal window after a successful message is sent

== Installation ==
From your WordPress dashboard

1. **Visit** Plugins > Add New
2. **Search** for \"Pop-up maker for CF7\"
3. **Install and Activate** Pop-up maker for CF7 from your Plugins page
4. On the editing page of your forms will appear new tab that will allow you to set the look of your messages
